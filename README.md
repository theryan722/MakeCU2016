# Myo and Furious
MAkeCU 2016 Hackathon Project
# Controls
Accelerate: Fist Hand
Decelerate: Fingers Spread
Turn Left: Turn hands in steering wheel form to left
Turn Right: Turn hands in steering wheel form to right
